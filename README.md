# Тест на проверку создания нового заказа в Яндекс.Самокат с помощью API

## Описание проекта
Автотест для проверки API сервиса аренды самокатов. Тест создает заказ и проверяет возможность получения данных о заказе по его трек-номеру.

## Тест-кейс
1. Отправить запрос на создание заказа
2. Сохранить номер трека из ответа
3. Отправить запрос на получение заказа по треку
4. Проверить, что код ответа равен 200

## Структура проекта
- `configuration.py` - конфигурационные данные (URL сервера, эндпоинты)
- `data.py` - тело запроса для создания заказа
- `sender_stand_request.py` - функции для отправки запросов к API
- `test.py` - файл с автотестом
- `SQL.txt` - SQL-запросы для заданий 1 и 2
- `requirements.txt` - зависимости проекта

## Требования
- Python 3.7 или выше
- Установленные пакеты: pytest, requests
